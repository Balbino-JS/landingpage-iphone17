📱 iPhone 17 Pro – Landing Page (Projeto Front-end)

Este projeto é uma Landing Page moderna e responsiva, inspirada no site oficial da Apple, desenvolvida com foco em boas práticas de front-end, componentização, responsividade e experiência do usuário.

🧠 Projeto criado com objetivo educacional, seguindo uma abordagem didática semelhante à utilizada por Rodolfo Mori (DevClub), priorizando clareza, organização e aprendizado prático.

🚀 Tecnologias Utilizadas
⚛️ React.js

Utilizado para construir toda a interface do projeto de forma componentizada, facilitando:

Reaproveitamento de código

Organização do projeto

Manutenção e escalabilidade

Cada seção do site foi separada em componentes, como:

NavBar

Hero

Intro

Highlights

Colors

Footer

💡 Pensamento muito usado pelo Rodolfo Mori: “quebrar tudo em componentes pequenos e reutilizáveis”.

🎨 Tailwind CSS

Framework CSS utilitário usado para estilização rápida e eficiente.

Principais vantagens:

CSS direto no JSX

Alta produtividade

Layouts responsivos com facilidade

Código limpo e organizado

Utilizado para:

Responsividade (md:, lg:)

Cores, gradientes e sombras

Animações (hover, transition, scale)

Layout com flex e grid

🧠 React Hooks (useState)

Utilizado para controle de estado, principalmente na seção de cores do produto.

Exemplo de uso:

Alterar a imagem do iPhone conforme a cor selecionada

Aplicar borda branca no botão da cor ativa

🎯 Aqui entra um conceito fundamental ensinado no DevClub: estado controla o que o usuário vê na tela.

🖼️ Manipulação de Imagens

As imagens do projeto foram usadas para simular:

Produto

Destaques de design

Câmeras

Processador

Com uso de:

object-cover

absolute / relative

z-index

Gradientes para melhor contraste visual

📱 Responsividade

O projeto é totalmente responsivo, adaptando-se a:

Mobile

Tablet

Desktop

Utilizando:

Grid responsivo

Flexbox

Breakpoints do Tailwind (md, lg)
